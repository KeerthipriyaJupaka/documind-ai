# DocuMind AI 🚀

## Overview
DocuMind AI is an AI-powered technical documentation generator designed to help developers automatically generate project documentation from GitHub repositories.

It simplifies onboarding, improves documentation consistency, and saves developer time using AI-generated documentation.

---

## Features

- AI-generated project documentation
- README generation
- API documentation support
- Pull request summarization
- Deployment notes generation
- Modern responsive UI

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js

### AI Integration
- Gemini API

## Installation

### Clone Repository

```bash
git clone https://github.com/KeerthipriyaJupaka/documind-ai.git
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
node server.js
```

---

## Usage

1. Enter a GitHub repository URL
2. Click "Generate Docs"
3. AI generates project documentation automatically

## Problem Statement
